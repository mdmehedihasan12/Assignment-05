What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programs and scripts to access and manipulate the content, structure, and styles of web pages.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to handle events on a parent element instead of directly attaching event listeners to individual child elements. This is particularly useful in scenarios where you have multiple child elements that require similar event handling.
